Simular Word Game

Description

in this game there are around 60 words there. you need to find a similar word for each word. you can click twise at the time. if you select correct word, "You selected Correct Word" message will appear. if you click the same image, "You have clicked the same image!" message will appper. if you select all images, "Congratulations! You found them all!" message will appear. there are no time limits for this game. if you select wrong image, "You selected Incorrect Word" message will appear. take your own time to play the game and get knowledge about similar words.
